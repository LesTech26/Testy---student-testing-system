# Testy---student-testing-system
Testy — веб-система для онлайн-тестирования студентов с разделением ролей (Студент, Преподаватель, Администратор). Поддерживает гибкую настройку тестов, защиту от списывания, автоматическую проверку и детальную статистику. Идеально подходит для учебных заведений с многофилиальной структурой (поддержка городов).
